Lesson 1: The Basics of the Web and HTML

HTML Basics

HTML stands for Hyper Text Markup Language. which is what defines most web pages.

HTML is made up of the following...
Text content shows what you see.
Markup shows what your page will look like.
References to other documents, such as images are possible.
Links to other web pages are possible too.

Tags

HTML uses tags in a certain structure to create a webpage, such as !doctype, head, title, body, etc.
Tags are what HTML uses to mark text in a certain way.
A tag normally has an opening tag, text, then a closing tag.
Tags that do not contain content are called "void" tags.
I learned how to use bold and to provide emphasis via the italics tags.

Inline vs. Block

Tags can be either inline or block.
Inline tags examples are break, bold, emphasis, void tags, new lines, etc.
Block tags create an invisible block around the content. Examples of this is paragraph and div.

Attributes

Attributes, ATTR equal a vallue.
The anchor tag creates links to other websites.
I can add images using the image tag and src attribute.

Lesson 2: Creating a Structured Document with HTML

Developer Knowledge

A Webpage uses structure, css, and java scripts. Structure is like the walls of your house that defines rooms. CSS shows the style, much like paint or decorations in a room. JS or Java Script creates the interactive components, much like the remote control or a garage door opener in your house.

Developer Tools show the structure of a webpage.
DOM - Document Object Model is the tree like structure most commonly used today.
Rectangles are used to represent any element, even if it looks like something else. These rectangles are used on an imaginary grid to account for the space on a page.

Codepen, Scratchpad, Sublime Comparison

Codepen gives 2-3 seconds feedback, early sharing capabilities (but requires an account), allows HTML and CSS, and is considered to be somewhat professional.

Scratchpad gives immediate feedback, suitable for beginners, no sharing capabilities, allows for HTML, CSS, JS considered to have a low professionalism rating.

Sublime gives feedback on refresh only, difficult to use, hard to share, supports all languages, uses your computer's browser, considered to be the most professional.

Lesson 3: Adding CSS Style to HTML Structure

CSS Cascading Style Sheets

CSS- Cascading Style Sheets avoid program repitition. A CSS file can be used via the link tag. Or the same thing can be accomoplished via the style HTMl tag.

Inheritance is the use of DOM to relay properties between ancestors and descendents. Descendents can get text based properties, but not box-related properties. Inheritance cuts down on repitition, uses less CSS, faster web page loading, saves money on bandwidth and developmental costs.

Text based examples are color, font, line spacing, line height, list style, text align, text indent, visibility, etc.

Box based examples are background, border, display, float and clear, height and width, margin, outline, overflow, padding, position, etc.

Selector decides what style gets applied. It is defined by a class name, along with attributes and values.

Box Model consists of a margin, border, padding, and content. Padding can be adjusted by background. Border can be adjusted by the color property. Margin is the space between boxes. Width consists of content + left and right border + left and right margin.
